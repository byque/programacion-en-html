Enlaces HTML - Hiperenlaces

Los enlaces en HTML son hiperenlaces. Un hiperenlace es un texto o una imagen en la que se puede hacer click y saltar a otro documento.

En HTML los links se definen con el tag a.
Manos a los Cables

El atributo href especifica la dirección de destino (http://manosaloscables.wordpress.com/).

El texto del link es la parte visible (Manos a los Cables).

Al hacer click en el texto del link se abrirá el documento de la dirección especificada. No es obligatorio que el link tenga texto, puede ser una imagen HTML o cualquier otro elemento de HTML.

Sin un slash al final de las direcciones de subcarpetas, pueden generarse dos peticiones al servidor. Muchos servidores automáticamente aumentan el slash final a la dirección, y luego crean una nueva petición.

Enlaces Locales

El ejemplo anterior usaba una dirección URL absoluta (una dirección web completa).

Un enlace local (enlace al mismo sitio web) se especifica con una URL relativa (sin http:wwww...).
