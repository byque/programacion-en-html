Fuentes de CSS

La propiedad color define el color del texto del elemento HTML.

La propiedad font-family define la fuente usada por el elemento HTML.

La propiedad font-size define el tamaño del texto del elemento HTML.

El Modelo de Cuadro de CSS

Todo elemento de HTML tiene un cuadro a su alrededor, incluso si no se ve.

La propiedad border de CSS define un borde visible alrededor del elemento.

La propiedad padding de CSS define el espacio entre el borde y el texto.

La propiedad margin de CSS define el espacio entre el margen y el borde.

El Atributo id

Para definir un estilo especial para un elemento especial primero se añade un atributo id al elemento y luego se define un estilo para el elemento con la id especificada.

Soy diferente

El Atributo Class

Para definir un estilo para un tipo (class) específico de elementos, añadir un atributo class al elemento.

Soy una notificación de error!

Usar id para referirse a un elemento singular. Usar class para referirse a un grupo de elementos.
