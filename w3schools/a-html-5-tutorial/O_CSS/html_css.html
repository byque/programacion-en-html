Estilizado Interno

Es usado para definir un estilo para una página HTML. Se define en la sección head de una página HTML, dentro de un elemento style. Los párrafos y los encabezados h2 están configurados con estilo interno.

Este es un encabezado verde

Estilizado Externo

Una página de estilizado externa define el estilo para muchas páginas. Se puede cambiar el aspecto de un sitio web completo cambiando una página de estilizado externa. Para usarla hay que añadir un link a ésta en la sección head de la página HTML.

Los encabezados h1 están configurados con estilizado externo.
