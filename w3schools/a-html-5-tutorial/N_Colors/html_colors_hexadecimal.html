En HTML los valores RGB también pueden ser especificados usando valores de color exadecimal de la forma: #RRGGBB.

Color de fondo configurado rojo con #FF0000.

Color de fondo configurado violeta con #FF00FF.

Color de fondo configurado amarillo con #FFFF00.

Color de fondo configurado verde con #00FF00.

Color de fondo configurado azul con #0000FF letras blancas.

Color de fondo configurado cian #00FFFF.

La escala de grises se define habitualmente usando valores iguales para los valores #RRGGBB.

Color de fondo configurado negro con #000000 con letras blancas.

Color de fondo configurado gris con #808080.

Color de fondo configurado gris oscuro con #A8A8A8.

Color de fondo configurado gris claro con #D3D3D3.

Color de fondo configurado blanco con #FFFFFF.
