HTML kbd define entrada de teclado.

HTML samp define un ejemplo de un resultado de un programa de computadora.

HTML code define código de programación.

Code no preserva los espacios y saltos de línea adicionales. Para que lo haga, colocar code dentro de pre.

		
		var x = 5;
		var y = 6;
		document.getElementById("demo").innerHTML = x + y;

HTML var define una variable. La variable puede ser de en una expresión matemática o de un programa.
